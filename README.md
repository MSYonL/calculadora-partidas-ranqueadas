 🎮 Calculadora de Partidas Ranqueadas

Este é o segundo desafio de projeto do curso de Lógica de Programação da **DIO (Digital Innovation One)**. O objetivo foi criar uma lógica funcional para classificar o nível de um herói com base no seu saldo de vitórias.

## 🛠️ Tecnologias Utilizadas
* **JavaScript**: Linguagem principal para a lógica do sistema.
* **Node.js**: Ambiente de execução para os testes via terminal.
* **VS Code**: Editor de código utilizado no desenvolvimento.

## 📝 O que foi praticado:
- **Variáveis**: Armazenamento de dados dos jogadores.
- **Operadores**: Cálculos matemáticos para obter o saldo de vitórias.
- **Estruturas de Decisão**: `if`, `else if` e `switch` para classificar o nível.
- **Estruturas de Repetição**: `for` para percorrer tabelas de dados de forma dinâmica.
- **Funções**: Encapsulamento da lógica para tornar o código reutilizável.

## 🚀 Como funciona
A função recebe a quantidade de vitórias e derrotas de um jogador, calcula o saldo e retorna o rank correspondente:

| Vitórias | Nível |
| :--- | :--- |
| Menor que 10 | Ferro |
| Entre 11 e 20 | Bronze |
| Entre 21 e 50 | Prata |
| Entre 51 e 80 | Ouro |
| Entre 81 e 90 | Diamante |
| Entre 91 e 100 | Lendário |
| Maior ou igual a 101 | Imortal |
